Prueba Backend Django – INACAP

Proyecto backend desarrollado en Django

Desc
La aplicación implementa un sistema backend con arquitectura Django, utilizando base de datos QLite, manejo de rutas, vistas y plantillas, cumpliendo con los requerimientos solicitados en la evaluación.
Tecnologías
- Python
- Django
- SQLite
- HTML / CSS
- Git & GitHub

Funcionalidades
- Configuración de proyecto Django
- Gestión de aplicaciones internas
- Operaciones CRUD
- Manejo de vistas y templates
- Persistencia de datos con SQLite
